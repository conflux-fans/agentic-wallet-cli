# Agentic Wallet

本项目是一个 Agentic EVM Wallet, 它最大的特点是用户可以通过自然语言的方式与钱包进行交互. 本项目主要用于探索 Agent 开发, 因此是一个示例项目.

## 产品形态

这是一个 cli 程序, 为用户提供了一个命令行界面，通过自然语言与钱包进行交互。

## 账户私钥

私钥储存不是项目的重点, 可以通过环境变量配置. 但在实际生产项目中, Agent 不能直接处理私钥.

## 支持的功能

1. 支持多链, 首批支持 Conflux eSpace 和 Monad
2. Native 操作
   1. 查看账户基本信息: 余额, nonce
   2. 进行 native 资产转账
   3. 交易历史: 需要 index API
3. ERC 20 操作
   1. 每条链有预置的 token 列表(usdt, usdc), 支持直接指定 token 地址
   2. 查询 ERC20 资产余额
   3. 进行 ERC20 资产转账 或 approve 操作
   4. erc20 transfer 历史: 需要 index API
4. Uniswap V2 操作
   1. 查询流动性池信息 (token 对、流动性、价格)
   2. 进行代币交换 (swap)
   3. 添加/移除流动性
5. LayerZero 跨链
   1. 查询跨链路由信息 (费用)
   2. 发送跨链转账 (跨链转移资产)
   3. 查询跨链交易状态
6. 支持通过 skill 增加对其他协议的支持(第一步先不实现)

## Agent 能力

1. 自动从用户输入中理解意图和执行动作
2. 能够判断用户信息的缺失, 并通过多轮对话的方式补全
3. 某些操作可能是分两步完成的, 比如跨链转账需要现在原链发送交易, 目标链claim token
4. 支持用户多步操作的需求: 换 100 usdt 为 cfx, 然后转给用户 b.

## 技术栈

1. TS 语言
2. 使用 vercel 的 ai SDK
3. 大模型使用 openrouter 作为 provider
